# FAQ Accordion Page

A responsive FAQ accordion page built with **React** and **CSS** to practice reusable components, state management, and clean UI interaction.

## Overview
This project was created to strengthen my understanding of React fundamentals, especially component composition, props, and state handling using `useState`.

## Features
- Expand/collapse FAQ items
- Single-open accordion interaction
- Clean and responsive layout
- Component-based React structure

## What I Practiced
- Reusable React components
- Props-based data flow
- State management with `useState`
- UI interaction logic for accordion behavior

## Tech Stack
- React
- JavaScript
- CSS

## Run Locally
```bash
npm install
npm start
```
